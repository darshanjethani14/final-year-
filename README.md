## IELTS Preparation Web Application

This is a professional IELTS preparation interface built with **React**, **React Router**, **Tailwind CSS**, and **Vite**. It is designed to resemble modern IELTS preparation platforms and can be extended into a full product by wiring it to your backend and test engine.

### Tech Stack

- **React** functional components
- **React Router** for navigation
- **Tailwind CSS** for styling
- **Vite** for the build tool

### Project Structure

- `src/components`
  - `Navbar.jsx`
  - `Sidebar.jsx`
  - `ModuleCard.jsx`
  - `QuestionCard.jsx`
  - `Timer.jsx`
- `src/pages`
  - `Home.jsx`
  - `Login.jsx`
  - `Dashboard.jsx`
  - `Modules.jsx`
  - `Test.jsx`
  - `Results.jsx`

### Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the printed local URL (usually `http://localhost:5173`) in your browser.

You can now customize questions, connect authentication, store real results, and integrate your own backend APIs while keeping this UI as the foundation.
